# Off-Page SEO Execution Playbook (February 21, 2026)

## Goal
Earn relevant authority links and mentions that reinforce Home Harmony's topical expertise across family operations.

## Weekly Execution Sprint
1. Pick one theme cluster:
   - meals/grocery
   - chores/tasks
   - workouts/macros
   - sleep/lifestyle
2. Build 30 qualified prospects.
3. Send 30 personalized emails (10/day for 3 days).
4. Publish one support asset that is easy to cite.
5. Run two follow-up rounds.

## Offer Types That Convert
- guest guide with implementation checklist
- actionable framework adapted to publisher audience
- data-backed insight post (what works in real households)
- expert quote contribution with practical examples

## Prospect Qualification
A target is valid when:
- audience overlaps parents/families/household operators
- editorial quality is reasonable
- topical relevance to a specific Home Harmony cluster exists
- likely external linking behavior exists

## Link Targeting Priority
1. Hub pages: strongest topical anchors.
2. Cluster detail pages: intent-specific reinforcement.
3. Resource page: broad brand discovery.

## Anchor Strategy
- Mostly natural branded + topical anchors.
- Avoid over-optimized exact-match repetition.
- Mix generic, brand, and phrase-match anchors.

## Reporting
Track in a sheet:
- prospect
- contact date
- follow-up dates
- result (won/lost/pending)
- linked URL
- anchor text
- domain quality notes

## Guardrails
- No spammy paid link schemes.
- No irrelevant directory blasting.
- Focus on relevance, editorial fit, and audience alignment.
