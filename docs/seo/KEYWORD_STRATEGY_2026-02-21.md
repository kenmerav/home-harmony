# Home Harmony Keyword Strategy (February 21, 2026)

## Core Positioning
Home Harmony is a family operating system, not just a meal app.

## Primary Keyword Clusters

### Cluster A: Meal + Grocery
- family meal planner app
- weekly meal plan for families
- shared grocery list app
- grocery list from meal plan
- pantry meal ideas

### Cluster B: Chores + Household Tasks
- family chore system
- chore tracker app
- household task tracker
- shared family to do app
- household management app

### Cluster C: Workouts + Nutrition
- workout tracker app
- strength training tracker
- family fitness planner
- macro tracking app
- high protein meal plan for families

### Cluster D: Lifestyle Tracking
- family sleep tracker
- period tracking planner
- alcohol habit tracker
- family wellness dashboard
- habit tracking system for families

## Intent Layers
- TOFU: templates, frameworks, how-to queries
- MOFU: app comparison and workflow queries
- BOFU: “best app”, “for families”, “shared”, “all-in-one” variants

## Mapping Rule
Every SEO page should map to:
1. One primary query intent
2. One adjacent supporting query
3. One conversion path (signup/demo)

## Publishing Rule
Prioritize pages where:
- search intent is specific
- implementation guidance is concrete
- internal links naturally connect to 2-4 adjacent clusters
